## Installation

### Prerequisites

[Node.js](https://nodejs.org) - 建議版本 v3.4.2 以上。

### Setup

Development:

``` bash
# install dependencies
$ npm install

# serve with hot reload at http://localhost:8080
$ npm run dev
```

Build Production:

``` bash
# build for production with minification in the dist folder
$ npm run build
```
